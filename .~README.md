[Creating EKS CLUSTER](https://docs.aws.amazon.com/eks/latest/userguide/getting-started-eksctl.html)

```
aws eks create-cluster \
--name capstone \
--node-type t2.micro \
--nodes 3 \
--ssh-access true\
--ssh-public-key pipeline.pub \
--managed
```
